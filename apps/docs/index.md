## Quick Start

```bash
# Clone the repository
git clone https://github.com/ilrein/writtencraft-ai.git

# Install dependencies
bun install

# Set up the database
bun db:push

# Start development servers
bun dev
```

Visit `http://localhost:3001` to start writing!
